Projects Done

INVGMT: Construction Site Inventory Management and Labour Tracking

INVGMT is a comprehensive web application developed using Django and Microsoft SQL Server. Specialized for construction sites, it efficiently manages inventory and tracks labour activities. The key features include:

Inventory Management:

INVGMT facilitates seamless tracking of construction site inventory. It manages the inflow of items to the site, tracks issuance to laborers, and records the return of unused items. The system provides detailed records of items used, promoting efficient resource utilization.

Tenant-Based Structure:

Utilizing Django's tenant structure, INVGMT ensures a secure and isolated environment for each construction project. This enhances data privacy and organization, allowing for effective management of multiple construction sites.

Labour Tracking:

The web app includes a robust labour tracking system, estimating project completion time based on the current workforce. It contributes to effective project management by providing insights into the labor activities.

Dashboard Analytics:

The dashboard utilizes JavaScript graphs to visually represent data. It displays records of items used, total spending on labour and inventory, and provides insightful graphs illustrating item trends and monthly spending patterns.

INVGMT stands as a testament to my proficiency in web application development, Django, Microsoft SQL Server, and my ability to address specific industry needs. The project showcases an organized and secure solution for construction site inventory and labour management.
